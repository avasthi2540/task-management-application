# Task Management Application

A web-based task management application built using HTML, CSS, and JavaScript. The application allows users to create, update, and delete tasks while storing data in the browser using Local Storage.

## 📌 Project Overview

This project helps users organize and manage daily tasks through a simple and user-friendly interface. Tasks remain available even after refreshing or reopening the browser.

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Store tasks using Local Storage
* Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage API

## 📂 How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Start adding and managing tasks.

## 💡 Example Functionality

* Create a new task
* Update task details
* Delete completed tasks
* Automatically save tasks in browser storage

## 🎯 Objective

To develop a simple task management system that demonstrates front-end development skills and browser-based data persistence using Local Storage.
